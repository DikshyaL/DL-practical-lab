# Deep Learning Practical Labs

This repository contains weekly practical implementations and experiments for the course **COMP 488 - Neural Networks and Deep Learning**. The goal is to build a strong foundation in deep learning by combining theoretical concepts with hands-on coding using Python, NumPy, and PyTorch.

---

##  Objectives

- Understand core deep learning concepts through implementation  
- Gain hands-on experience with tensor operations and linear algebra  
- Explore neural network fundamentals and training techniques  
- Analyze numerical stability and optimization challenges  
- Build and evaluate simple machine learning and deep learning models  

---

## 🛠️ Tools & Technologies

- Python  
- NumPy  
- PyTorch  
- Colab
